About the developer

Eduardo Perez

Developer of the Bay Area Air Quality Episode Finder.

Back to application

This independent Earth-observation software project was created as a public technical portfolio project for consideration for the NASA EarthRISE Developers Academy internship opportunity. It analyzes Sentinel-5P tropospheric NO₂ column data for the San Francisco Bay Area with the methodology, coverage, and limitations shown alongside every result.
