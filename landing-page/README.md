# Superseded: landing-page plan

This directory originally planned a custom-domain landing page linking
to a separately published Earth Engine App. That plan was superseded on
2026-07-18: Railway hosts the complete public application, and the
application code lives under `app/` (`app/backend/` and
`app/frontend/`; both deployed, see
[docs/architecture.md](../docs/architecture.md)). The Earth Engine App
form remains only a possible fallback.

This directory contains no application code. A simple landing or entry
page may still exist later as part of the application.
