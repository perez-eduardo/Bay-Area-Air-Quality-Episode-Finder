# Earth Engine app

**Status: not started.**

This directory will hold the Google Earth Engine app — the project's main
runtime for geospatial analysis and the public-facing dashboard.

**Decided:** the app will be built with the Earth Engine JavaScript API and
developed and published through the Earth Engine Code Editor.

Still blocked on the remaining Phase 0 decisions (see
[docs/roadmap.md](../docs/roadmap.md)): the Bay Area region definition and
evaluation/approval of a first dataset.

Planned first milestone (Phase 1): Bay Area map, indicator selector, basic
time series, placeholder episode summary, and visible methodology/limitations
notes.
